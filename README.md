# Portfólio Bryan / bps2414

Portfólio pessoal em Next.js para apresentar projetos selecionados com foco em desenvolvimento web, interfaces, ferramentas locais e pequenos negócios.

## Projetos principais

- **ChamadaFácil**: sistema web de chamados para pequenos negócios, com abertura pública de solicitações, consulta por código e e-mail, painel administrativo protegido, Supabase Auth, PostgreSQL, RLS e Server Actions.
- **Prado Auto Peças**: site institucional publicado para negócio local, com foco em clareza, contato rápido e manutenção simples.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- next-themes
- lucide-react

## Comandos

```bash
npm run dev
npm run lint
npm run build
```

## Observações

O conteúdo do ChamadaFácil foi adaptado a partir do repositório real do projeto, incluindo README, documentação técnica, estudo de caso, migrations Supabase, rotas públicas e administrativas, além de assets próprios para portfólio.
